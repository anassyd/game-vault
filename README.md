# GameVault

A modern video game discovery app built with React, Vite, shadcn/ui, and Framer Motion. Uses the RAWG API.

## Setup

```bash
npm install
```

Create a `.env` file based on `.env.example` and add your RAWG API key (get one at https://rawg.io/apidocs).

```
VITE_RAWG_API_KEY=your_key_here
VITE_RAWG_BASE_URL=https://api.rawg.io/api
```

## Run

```bash
npm run dev
```

## Build

```bash
npm run build
```

## Stack

- React 19 + TypeScript
- Vite
- Tailwind CSS v4
- shadcn/ui
- Framer Motion
- React Router
- Axios
- RAWG API
